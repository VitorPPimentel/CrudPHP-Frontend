# Vue.js Project - Project CRUD System

## Overview

This is a Vue.js project that implements a simple CRUD (Create, Read, Update, Delete) system for project registration. The main purpose of this system is to allow users to:

- Create new projects
- View information about existing projects
- Update project details
- Delete existing projects

The project has been developed as a sample application to demonstrate the use of Vue.js in building interactive web applications.

## Installation

Before you begin, make sure you have Node.js and npm (Node Package Manager) installed on your machine. If you don't have them installed, you can download them from the official website: [Node.js](https://nodejs.org/).

After installing Node.js, follow these steps to set up the development environment:

1. Clone the repository:

   ```bash
   git clone https://github.com/luidi64/CrudPHP-Frontend.git
   ```

2. Navigate to the project directory:

   ```bash
   cd frontend
   ```

3. Install project dependencies:

   ```bash
   npm install
   ```

## Usage

To start the application locally, use the following command:

```bash
npm run serve
```

This will launch the development server, and you can access the application in your browser at [http://localhost:8080](http://localhost:8080).
